Keyboard Shortcuts
------------------

Keys h, j, k, l are used for moving as in Vim:

- ``h`` - left;
- ``j`` - down;
- ``k`` - top;
- ``l`` - right.

- ``10h`` means repeat the "move to left" command ten times;

- ``+`` - zoom in;
- ``-`` - zoom out;
- ``0`` - reset scaling;

- ``Escape`` - close sidebars;

- ``/`` - focus the search field.
