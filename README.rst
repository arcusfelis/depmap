Keyboard Shortcuts
------------------

Keys h, j, k, l are used for moving as in Vim:

- ``h`` - left;
- ``j`` - up;
- ``k`` - down;
- ``l`` - right.


- ``10h`` - repeat the "move to left" command ten times;


- ``+`` - zoom in;
- ``-`` - zoom out;
- ``0`` - reset scaling;


- ``Escape`` - close sidebars (or just click on them);


- ``/`` - focus the search field.
